    Take your decision-making to the next level with the Tradeoff Analytics user interface! Using a guided decision process, novel interactive visualizations, and comparison tools, you can fully understand the trade-offs between different options, ensuring you make an informed decision.

    The Tradeoff Analytics UI is available as a Javascript library and may be embedded seamlessly in any web application, in any domain. View documentation.

    You can also bring your own data using the Tradeoff Analytics CSV to JSON converter.
